A Serious Game

A serious game application implemented on Android Studio to support people that suffer from anxiety and depression. The game is visual novel, giving the player the option to select which story they wish to experience. The story they select is then played in the application, where they can read through it and they are presented with Cognitive Behavioural Therapy challenges in specific points in the story, and they have to complete these challenges to progress with the story.
This project was implemented for my undergraduate project

Click here for my thesis.

What is a serious game?
A serious game is a game implemented for (you guessed it) a serious purpose. That purpose can be educational, health, training, and so much more. You are essentially combining a tedious task—medicine—with an element of entertainment—a spoonful of sugar.
